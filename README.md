## 🔧 What I Built

### 1. Internal Data Automation
- Built a self-service data extraction page, replacing manual SQL report delivery to other departments
- Reduced recurring data requests by enabling non-technical staff to query and export data independently

### 2. Operations Management Tools
- Developed teacher organization management pages, allowing the operations team to handle restructuring directly without developer involvement
- Removed bottleneck of routing all DB changes through the dev team

### 3. CRM & Internal Systems (MGT/CRM)
- Added real-time student learning progress tracking to the CRM dashboard
- Built account permission management page for role-based access control
- Developed a teacher-parent consultation memo feature
- Created a DB schema reference tool for developers to browse table structures and descriptions

### 4. Learning Player (Front Office)
- Developed the video learning player for the MilkT tablet app using **ExoPlayer / NodePlayer**
- Integrated CDN-delivered media content from the media team into the playback interface
- Connected the post-video question bank flow, querying student responses against the DB
- Wrote API documentation and collaborated across the media, content, and QA teams

### 5. Security (ISMS Compliance)
- Applied data masking to sensitive fields ahead of annual ISMS certification audits
