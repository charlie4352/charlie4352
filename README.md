## What I Built

### 1. Internal Data Automation

- Reduced recurring data requests by enabling non-technical staff to query and export data independently

### 2. Operations Management Tools
- Developed teacher organization management pages, allowing the operations team to handle restructuring directly without developer involvement
- Removed bottleneck of routing all DB changes through the dev team

### 3. CRM & Internal Systems (MGT/CRM)
- Added real-time student learning progress tracking to the CRM dashboard

- Developed the video learning player for the MilkT tablet app using **ExoPlayer / NodePlayer**
